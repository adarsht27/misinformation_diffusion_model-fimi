# Docs

This folder contains the written report for this project and its potential applications in combating misinformation spread by foreign interference among different societies.

## File

### `ABM project.pdf`
Full project report covering:
- Policy context — EU FIMI threat landscape and EEAS mandate
- Model framework — SBFC mechanics and NetLogo implementation
- Critique of the original model and rationale for extensions
- Scenario design and parameter calibration
- Simulation outcomes for Scenario 1 (Vulnerable) and Scenario 2 (Resilient)
- Assessment of model validity and limitations
- Policy recommendations for EEAS STRAT.1 and STRAT.4
